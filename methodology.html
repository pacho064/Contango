Methodology

How numbers are sourced, calculated, and labeled for trust, reproducibility, and future paid analytics.

Source hierarchy
Tier 1
Official issuer / regulator publication (NOC statement, official report, exchange bulletin).
Tier 2
Official statistical body / intergovernmental publication (OPEC, EIA, JODI, port authority reports).
Tier 3
Professional market reporting used as a fallback (for context, not preferred for benchmark construction).
Numbers on public pages are indicative and tied to the publication cadence of the upstream source.
Core 5 formulas
Lead series
Arab Light OSP vs Brent spread = Arab Light reference value - Brent reference value.
Event baseline
For each series, baseline is stored at event date (currently 2026-02-01). Change = current - baseline.
Display logic
If no new official release is available, last confirmed value is retained with latest ingestion timestamp.

Quality policy
Provenance
Every displayed figure should map to a source name and source URL.
No synthetic prices
Public values are not randomly generated; they should represent latest stored observations.
Transparency
Formula definitions remain public for all free-tier headline metrics.
Paid tier direction
Future paid features should add depth (scenario dashboards, route stress, disaggregated flows), not hide core methodology.
